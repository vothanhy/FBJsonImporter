*** NOTE ***

There are 2 folders:

* FBJsonImporter is final release. 
Please copy the [wordpress]/wp-content/plugin and activate it.
Access the plugin at Settings->FB Json Importer

* FBJsonImporter_Test/FBJsonImporter is the same plugin but it was customized for testing purpose such as uploading 5000 posts, 10000 posts or 20000 post.
[How to test]
1. Just copy FBJsonImporter_Test/FBJsonImporter to /plugins folder
2. Activate it
3. Access the plugin from Settings->FB Json Importer
